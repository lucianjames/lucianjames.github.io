    GLSLSlime

    Complex behaviour from a simple set of rules, being simulated massively in realtime by utilising GPU compute. This project is a rewrite of an older project of mine, now with significantly cleaner code and a nice set of controls.

    The slightly strange name of this project comes from the fact that I originally found inspiration for this program from a video showcasing a similar simulation, which was based on the growth of slime mold.

    The details

    Each agent leaves a trail and turns towards the trails of all the other agents in the simulation. Tweaking turn speed, trail diffusion, agent sensor location, and the few other parameters of the simulation can produce an incredibly diverse range of behaviours. This simulation runs incredibly fast thanks to the use of compute shaders within OpenGL, I can run a simulation with 5,000,000 agents on a 4096x4096 grid at 60 frames per second on an RTX3090.
