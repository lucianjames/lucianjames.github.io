    Barebones OS dev

    While very far from being a complete and functional kernel, I have developed a bootable system that has the ability to draw text directly to the hardware framebuffer

    View on github

    Languages/software used

    C, NASM, GRUB

    Date of completion

    Still a work in progress!

    The details

    Based on resources from wiki.osdev.org amongst others, this kernel can boot and render text (or whatever you wish) to the screen via the framebuffer.
    While I am yet to implement real OS functionality (memory management, etc), development has already been a great insight into developing low level standalone software. The kernel is primarily written in C (without access to the standard library) with NASM used for the very start of execution after GRUB hands over to my code.
