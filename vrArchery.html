    Unity VR archery game

    Making this game was part of my IT qualification, it was produced for the oculus quest 2. Thanks to my previous experience with 3D software (mostly blender) I was able to create a VR game which I am pretty proud of despite having no previous experience developing for VR or even using unity.

    Languages/software used

    Unity, C#, Blender

    Date of completion

    This game along with the various bits of related coursework was completed around the end of may 2022

    The details

    This game is powered by Unity, and is built for the Oculus Quest 2. I had fun doing some 3D modelling in Blender for the first time in a while. My previous experience with 3D software was incredibly useful for this project, since my class was not given any good instruction on how these kinds of software work and behave.
